- Svelte Webviews
- [Markedjs](https://github.com/markedjs/marked)
- [Highlightjs](https://github.com/highlightjs/highlight.js/) with GitHub Dark Theme and [Blade Language](https://github.com/miken32/highlightjs-blade)
